Django中提供了”信号调度”,用于在框架执行操作时解耦.

一些动作发生的时候,系统会根据信号定义的函数执行相应的操作

对于Django内置的信号,仅需注册指定信号,当程序执行相应操作时,系统会自动触发注册函数

在相应的应用(app)目录下的__init__.py文件中进行定义,(可以另外创建一个.py文件, 再在__init__.py文件导入该文件)
