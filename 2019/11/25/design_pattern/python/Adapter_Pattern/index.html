适配器模式

介绍

将一个类的接口转换成客户希望的另外一个接口。适配器模式使得原本由于接口不兼容而不能一起工作的那些类可以一起工作。

应用场景：

  • 面对一个有全新接口的类库而又不能改变现有代码时，最先想到的做法是，在这两个系统之间添加一个适配器。
  • grpc 也可以认为是一种适配器，提供了跨语言调用能力
  • sqlalchemy 可以在不改变代码的情况下对接多种数据库
